# Air Canvas Project

## Overview
Air Canvas is a computer vision–based virtual drawing application that allows users to draw on a digital canvas using hand gestures captured through a webcam. The project uses real-time hand tracking to create a touchless drawing experience.

## Features
- Real-time hand tracking
- Draw using finger movements
- Multiple color options
- Eraser functionality
- Clear screen option
- Interactive virtual canvas

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How It Works
The webcam captures hand movements, and MediaPipe detects hand landmarks. The fingertip coordinates are tracked and used to draw on a virtual canvas in real time.
